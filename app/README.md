# GameHub

## Descripción

GameHub es una plataforma de videojuegos desarrollada con Next.js. Permite visualizar videojuegos obtenidos desde una API externa, ver el detalle de cada juego y guardar juegos favoritos usando Context API y localStorage.

## API utilizada

El proyecto consume datos de RAWG API, una API REST de videojuegos que permite obtener información sobre juegos, imágenes, fechas de lanzamiento, calificaciones y descripciones.
