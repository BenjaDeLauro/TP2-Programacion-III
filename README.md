# GameHub

## Descripción

GameHub es una plataforma web de videojuegos desarrollada con **Next.js**, **JavaScript** y **Tailwind CSS**.
Permite explorar juegos, ver detalles de cada título y guardar favoritos usando estado global y localStorage.

## Tema del proyecto

Plataforma de videojuegos.

## Funcionalidades

* Página principal.
* Catálogo de videojuegos.
* Buscador de juegos.
* Detalle individual de cada juego.
* Sistema de favoritos.
* Dashboard con perfil y configuración.
* Preferencias guardadas en localStorage.

## Rutas principales

* `/` - Inicio
* `/juegos` - Listado de videojuegos
* `/juegos/[id]` - Detalle dinámico de un juego
* `/favoritos` - Juegos favoritos
* `/nosotros` - Información del proyecto
* `/dashboard` - Panel principal
* `/dashboard/perfil` - Perfil
* `/dashboard/configuracion` - Configuración

## Requisitos cumplidos

* Tres rutas base.
* Una ruta dinámica.
* Una ruta anidada.
* Navbar y Footer.
* Layout principal y layout anidado.
* Navegación con `Link` de Next.js.
* Componentes reutilizables.
* Server Components y Client Components.
* Context global.
* Persistencia con localStorage.
* Estilos con Tailwind CSS.

## Instalación

Clonar el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

Entrar a la carpeta:

```bash
cd TP2-Programacion-III
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npm run dev
```

Abrir en el navegador:

```txt
http://localhost:3000
```
